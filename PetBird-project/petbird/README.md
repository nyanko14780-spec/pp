# 냥코 펫 (PetBird) 🐣

화면 위에 떠다니는 병아리 캐릭터 오버레이 앱이에요.
- 캐릭터를 드래그해서 화면 아무데나 옮길 수 있어요.
- 캐릭터를 탭(짧게 클릭)하면 랜덤 짤 이미지가 통통 튀어나와요.
- 가끔 저절로 움찔움찔 움직여요 (살아있는 느낌).

## 📱 GitHub 모바일 웹으로 APK 빌드하는 법 (컴퓨터 없이!)

1. 이 저장소의 모든 파일을 새 GitHub 저장소에 그대로 업로드하세요.
   (저장소 생성 → "Add file" → "Upload files"로 폴더째 올리면 편해요)
2. `main` 브랜치에 push 하면 자동으로 Actions가 실행돼요.
   - 저장소 상단 **Actions** 탭 → "Build APK" 워크플로우 확인
   - 수동으로 실행하려면 Actions 탭 → Build APK → **Run workflow** 버튼
3. 빌드가 끝나면 (초록 체크 ✅) 해당 워크플로우 실행 화면 맨 아래
   **Artifacts** 항목에서 `PetBird-debug-apk`를 다운로드하세요.
4. 폰에서 다운로드한 zip을 풀면 `app-debug.apk`가 나와요. 그걸 탭해서 설치하면 끝!
   (출처를 알 수 없는 앱 설치 허용이 필요할 수 있어요)

## 앱 사용법

1. 앱을 실행하면 "다른 앱 위에 표시" 권한 요청 화면이 떠요 → 허용해주세요.
2. "펫 시작하기" 버튼을 누르면 캐릭터가 화면 위에 떠다니기 시작해요.
3. 홈 화면이든 다른 앱이든 캐릭터가 계속 떠 있어요.
4. 캐릭터를 탭하면 랜덤 짤이 튀어나오고, 드래그하면 위치를 옮길 수 있어요.

## 🖼️ 짤 이미지를 내가 원하는 사진으로 바꾸고 싶다면

지금은 임시 짤 5개(`meme1.png` ~ `meme5.png`)가 들어있어요.
GitHub 웹/앱에서 `app/src/main/res/drawable/` 폴더로 들어가서
같은 이름(`meme1.png` 등, 정사각형 PNG 권장)으로 파일을 교체 업로드하면
다음 빌드부터 내가 고른 사진이 랜덤으로 튀어나와요.
개수를 늘리고 싶으면 `meme6.png`처럼 추가하고,
`OverlayService.kt`의 `memeImages` 목록에 `R.drawable.meme6`를 추가해주면 돼요.

## 캐릭터 이미지 교체

`app/src/main/res/drawable/character.png` 파일을 원하는 캐릭터 이미지로
같은 이름으로 교체하면 돼요. 배경이 투명한 PNG를 쓰면 자연스러워요.
